ABOUT THIS PROJECT

AI Teaching Assistant

An AI-powered semantic search engine built using Retrieval-Augmented Generation (RAG) to help students quickly locate concepts inside Deep Learning lectures.

Project Overview

This project was developed as a learning initiative to understand and implement Retrieval-Augmented Generation (RAG). Instead of searching videos using titles or keywords, the application performs semantic search over lecture transcripts, allowing users to ask natural language questions and instantly discover the most relevant lecture along with exact timestamps.

Learning Resource

All lecture transcripts used in this project are sourced from the excellent 100 Days of Deep Learning playlist by CampusX.
